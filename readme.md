# How to use Turtle_Fun_Sketch.py?

## Commands:

> draw_forward, ```Up```: Move forward/upward
> 
> draw_backward, ```Down```: Move backward/downward
> 
> draw_backward, ```Down```: Move backward/downward
> 
> turn_left, ```Left``` : Turn left
> 
> turn_right, ```Right``` : Turn right
> 
> clear_drawing, ```space``` : Clear the drawing
